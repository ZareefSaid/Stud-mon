About

Studémon is an amalgamation of Pokemon, studying and discord. Since many people are already spending time on discord it makes it more convenient to study there too! The Pokemon like battle makes studying more fun than boring old flashcards.

  • It includes easy to use commands.
  • There's also a help server in case you're confused!
  • And it's easy to set up and start using right away!

We hope you enjoy Studémon!
